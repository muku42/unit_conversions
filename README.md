# unit_conversions
CS294 Module 16: Test

Convert from:
* kilometers to miles
* miles to kilometers
* years to minutes
* minutes to years
* joules to foe [Foe (unit) on Wikipedia](https://en.wikipedia.org/wiki/Foe_%28unit%29)
* foe to joules

Usage:
`python main.py <some_number>`

To run the unit tests:
`python run_tests.py`
